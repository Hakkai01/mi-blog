Pantalla De Bienvenida

13 de julio, 2026

A diferencia del HTML como tal, que define lo que hay en una pagina, y el CSS que modifica el como se ve esa pagina, el JS o (JavaScript) define lo que hace, este mismo responde a las acciones del usuario y modifica el HTML y el CSS, en tiempo real, obviamente estamos hablando desde la parte tecnica.

Lo que yo quiero hacer ahora es que, cuando el usuario o la persona haga click en el boton entrar, este mismo desaparezca de la pantalla, un CSS puede cambiar las cosas cuando el mouse pasa por encima de este, pero no lo desaparece si alguien hiciera click en el, esa ya es la funcion del Java.

para entenderlo mucho mejor tenemos que divir el proceso en 3 partes, y modificar tanto el css como el index agregando 3 bloques de codigo, primero lo haremos con el index: (ojo, esto lo tenemos que agregar despues de la primer etiqueta del "body") 

 INDEX
"div class="bienvenida" id="bienvenida""
    "div class="bienvenida-contenido""
        "h1>BlueNotesDocumentación personal de un proceso de aprendizaje en desarrollo web."/p"

        "button onclick="entrar()""Entrar →"/button"
    "/div"
"/div"

Aca lo que estamos haciendo es crear lo que sera nuestra nueva pantalla principal, y lo que aparecera cada vez que una persona ingrese a nuestra pagina, pero a esto tenemos que agregarle tambien un estilo, sino quedaria solo como un pedazo de texto colgado en una pared blanca (HORRIBLE).

Bla bla bla, mucho texto, no voy a entrar en detalle de todo lo que agregamos aca, unicamente voy a decir que se declararon las variables, que seria todo lo que tiene ."nombre" para que modifique etiquetas especificas con la clase declarada y asi no modifique nada mas, eso ya lo veniamos haciendo hace rato, ahora solo nos faltaria una cosa, agregar el script de java, y lo vamos a hacer directamente en el index, asi como el primer bloque que pusimos.

"script"
    function entrar() {
        document.getElementById('bienvenida').classList.add('oculta');
    }
"/script"

Este bloque de codigo tenemos que colocarlo justo antes de que se cierre la etiqueta del "body", sino no servira nada de lo que hicimos, mas adelante agregare una animacion de texto, pero para eso se tiene que implementar una libreria llamada GSAP, y no me voy a matar la cabeza aun con eso. (cuando lo ponemos en un archivo .js, no hace falta escribir el "script" "/script")

Como Funciona el Script

13 de julio, 2026

En este bloque agregamos bastantes cositas, primero expliquemos parte por parte del codigo:

funcion entrar() -> Este pedazo define la funcion, es decir declara un bloque de codigo con nombre, y este mismo no hace nada hasta que es llamado por una accion.

document -> Representa toda la pagina que tenemos, es el punto de entrada para que el JS (JavaScript pa los panas) manipule cualquier cosa dentro del documento.

.getElementById('bienvenida') -> (puto nombre tan largo) Busca en la pagina algun elemento que tenga declarado un id='bienvenida' y lo coge, (jaja lo coge), literalmente lo agarra, es como decirle al JS, encuentreme especificamente a este fulano.

.classList.add('oculta') -> agrega la clase CSS "oculta" al div, y como en el CSS ya definimos una variable como .bienvenia.oculta entonces lo que hacemos es que con esta clase, el div, se vuelve transparente con una animacion de 0.5 segundos, que tambien esta dentro de todo ese palabrerio que pusimos en el css.

Todo dentro del CSS es modificable, el chiste es jugar con los valores hasta que a uno le guste el resultado.

Ahora si podemos entrar en materia de lo que nos interesa, el java script, yo personalmente al principio lo odiaba, digamos que los profesores encargados de enseñarme este lenguaje de programacion lo hacian como la mrda, por ende le cogi mucho mas odio que amor, pero una vez uno se toma la molestia de ver cuales son sus capacidades, en el lugar que uno esta trabajando, pues aprende a quererlo igual, obvio no le hecho la culpa a los modos de enseñanza de los malparidos de mis profesores, pero en parte si.

Ahora, primero tenemos que entender 3 conceptos, los cuales ami no me explicaron tan claro, ojo estos aplicados al HTML y CSS, porque esa es nuestra base de trabajo:

1. Variables -> Almacenar/guardar informacion

let nombre = "Blueee"
let edad = 20
let aprendiendo = true

No hace falta explicacion, cada variable guarda la informacion que nosotros le indicamos, bien sea, texto, numeros, o incluso mas codigo.

2. Funciones -> Codigo con nombre que puede ser llamado

a diferencia de las variables que guardan informacion, las funciones ejecutan acciones, cuando uno lo solicita, y por lo general son bloques completos de codigo para una accion o instruccion especifica. 


function saludar() { 
    console.log("Hola!") 
} 

saludar() // esto la ejecuta

3. DOM -> (no el de marvel) Manipulacion de la pagina

// Encuentra un elemento por su id

let elemento = document.getElementById("mi-div")

// Cambia su contenido

elemento.innerHTML = "Texto nuevo"

// Agrega o quita clases CSS

elemento.classList.add("activo") elemento.classList.remove("activo")

La traduccion literal es muy sencilla, el DOM, como tal declara las acciones, bien sea algo como, encontrar, cambiar, agregar o borrar, antes lo habiamos usado para hacer la pantalla de bienvenida de la pagina, usamos algo asi:

document.getElementById('bienvenida').classList.add('oculta')

Que traducido al español nos diria algo como "Del documento x, traer el elemento que se identifique por el ID ('bienvenida') y agregarle la clase CSS ('oculta') ingles basico mi negro.
